# Jarvis

Simple productivity booster write with c++, qt.

### Install

First, clone the repo via git:

```bash
git clone https://github.com/SergiuszN/jarvis.git jarvis
```

And then compile.

```bash
$ sh run.sh
```

### Run
```bash
$ ./jarvis ${path}
```

> Note: requires a cmake version >= 2.6 and make
